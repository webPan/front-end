# 移动端

## 移动端适配讲了一下
* `rem`布局  
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```  
* `width=750`布局  
```html
<meta name="viewport" content="target-densitydpi=device-dpi,width=750,user-scalable=no">
```
